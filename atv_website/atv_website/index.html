ATV外贸网站系统

基于 PHP + MySQL 的专业外贸企业网站管理系统 v1.0

⚠ 本系统需要 PHP + MySQL 运行环境。当前为静态预览模式，完整功能需要部署到 PHP 服务器。

🔧 修复记录

  • 修复了 config.php 中 table() 函数重复定义导致的致命错误
  • 修复了 install.php 生成的配置文件缺少 table() 函数的问题
  • 优化了前台全屏自适应布局和视觉效果
  • 所有PHP文件已通过语法检查

🚀 安装步骤

  1. 将 /mnt/agents/output/atv_website/ 下所有文件上传到PHP服务器
  2. 在MySQL中创建一个空数据库
  3. 浏览器访问 http://您的域名/install.php
  4. 填写数据库信息，设置管理员账号，完成安装
  5. 安装完成后删除 install.php，登录后台 /admin/
